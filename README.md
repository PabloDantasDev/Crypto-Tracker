# Crypto-Tracker

## DOCUMENTAÇÃO:

- CRYPTO-TRACKER FOI FEITO PARA CONSULTAS SIMPLES DOS VALORES DAS CRYPTO-MOEDAS .
- CRYTO-TRACKER É ATUALIZADO AUTOMATICAMENTE DE FORMA DINÂMICA SEM A NECESSIDADE SE SER ATUALIZADA A PÁGINA.
